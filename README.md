# osaka-bay-ai-pinn

Physics-Informed Neural Networks (PINNs) for Osaka Bay Tidal Flow and Oil Spill Estimation using STOC LST data.

## 🌊 プロジェクト概要
本プロジェクトは、大規模災害（津波）発生時の迅速な意思決定支援を目的とした、**「流体物理学」と「ディープラーニング」を融合させたリアルタイム予測システム**です。

海洋数値シミュレータ（STOC）が出力する膨大なデータと、流体物理の法則（ナビエ・ストークス方程式の根底にある連続の式、および移流拡散方程式）をニューラルネットワークの損失関数に組み込む**PINNs（Physics-Informed Neural Networks）**を実装し、津波発生時における大阪湾の「流速場」および「オイルスピル（油拡散）」の挙動を高精度かつ高速に補完・予測します。

---

## 🛠 技術スタック & 実装の3大コア

本システムは、**【データ前処理（Fortran）】→【AIモデル構築（PyTorch）】→【視覚化（Matplotlib）】**の3つのパイプラインで構成されています。

### 1. 超高速データエンジニアリング (Fortran)
- **課題**: 海洋数値モデル（STOC）の出力するLSTファイルは、**5.6GBに及ぶ巨大なバイナリデータ**であり、通常のPythonスクリプトではメモリ不足やI/Oボトルネックが発生する。
- **解決策**: 低レイヤのメモリ管理と圧倒的な処理速度を持つ**Fortran**を用いて、バイナリから特定海域（Area 01）の流速・座標データを高速にパース・抽出する専用パイプラインを構築。

### 2. 物理情報ニューラルネットワーク (PyTorch)
データが存在しない「グリッドの隙間」や「未観測領域」を単に統計的に補間するのではなく、以下の**2つの流体物理の支配方程式（PDE）を損失関数としてネットワークに拘束（制約）**させています。

- **連続の式（質量保存則）**:
  $$\frac{\partial u}{\partial x} + \frac{\partial v}{\partial y} = 0$$
  予測される流速ベクトル $(u, v)$ が、流体として物理的に矛盾のない（水が突然消えたり湧き出たりしない）ダイバージェンスフリーな流れ場になるよう強制します。
- **移流拡散方程式（物質の移動・拡散）**:
  $$\frac{\partial C}{\partial t} + u \frac{\partial C}{\partial x} + v \frac{\partial C}{\partial y} = D \left( \frac{\partial^2 C}{\partial x^2} + \frac{\partial^2 C}{\partial y^2} \right)$$
  オイルの濃度 $C$ が、算出された潮汐流・津波流に乗って運ばれ（移流）、時間の経過とともに周囲へ広がっていく（拡散）というダイナミクスを正確にシミュレートします。

### 3. 動的ベクトルの可視化 (Python)
- 算出された連続的な流速場を `Matplotlib` の `quiver` プロット等を用いて2次元の潮流ベクトル図として視覚化。
- タイムステップごとのオイルの拡散濃度をヒートマップとして重ね合わせることで、直感的な被害予測情報の生成を可能にしました。

## 🚀 本手法の強み（AI×物理のブレイクスルー）
1. **次元の呪いとデータ不足の克服**
   従来の数値シミュレーションのように膨大な格子（メッシュ）を切ることなく、ニューラルネットの自動微分（Autograd）を用いて座標 $(x, y, t)$ から直接流速と濃度を算出するため、グリッドに依存しない連続的な解析が可能です。
2. **高い一般化性能と頑健性**
   AIモデル内部に物理法則（PDE）のトポロジーを保持しているため、データが極端に少ない海域や、津波のような急激な構造変化が起きる大ショック期のデータに対しても、物理的に破綻しない美しい収束（理想値までのロス低下）を実現しています。
